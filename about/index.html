À propos de TrainRando

TrainRando est un projet open-source qui répertorie des itinéraires de randonnée entre gares sur les sentiers GR en France. L'objectif : rendre la randonnée accessible sans voiture, en reliant les sentiers de Grande Randonnée au réseau ferroviaire.

Comment ça fonctionne

Un catalogue de 176 sentiers GR est enrichi automatiquement via OpenStreetMap : le pipeline identifie les gares SNCF et arrêts de bus à proximité de chaque sentier, génère un profil altimétrique et référence les hébergements (hôtels, campings) le long du tracé. Les arrêts de bus sont enrichis avec les noms de lignes GTFS pour faciliter la planification.

Sources de données

  • OpenStreetMap — tracés des sentiers GR, gares et données d'hébergement (licence ODbL)
  • SNCF Open Data — base de données des gares ferroviaires
  • transport.data.gouv.fr — données GTFS des arrêts de bus et noms de lignes
  • SRTM — données d'élévation pour les profils altimétriques
  • OpenTopoMap — fonds de carte topographiques
  • IGN Géoservices — fonds de carte Plan IGN

Les traces sont générées automatiquement et peuvent contenir des imprécisions. Vérifiez toujours votre itinéraire avant de partir en randonnée.

Open source

Le code source est disponible sur GitHub. Les contributions sont les bienvenues.
