Privacy policy

No personal data collection

TrainRando does not collect, store, or share any personal data. No user account is required to use the site.

No cookies

This site does not use any tracking cookies or analytics. The only data stored locally is your theme preference (light/dark), via your browser's localStorage. This information never leaves your device.

Static site

TrainRando is a fully static site. There is no server-side processing, no database, and no backend API that would process your data.

Third-party services

Map tiles are provided by OpenTopoMap and IGN (Géoportail). When you view a map, your browser sends requests directly to these services to load map tiles. These providers may receive your IP address as part of these requests.

Hosting

This site is hosted on GitHub Pages. Standard web server access logs (IP address, user-agent, pages visited) may be collected by GitHub in accordance with their privacy policy.

Contact

For any questions about this privacy policy, you can open an issue on the project's GitHub repository.
